# R Data Visualization Reports

This project is designed for generating data visualization reports using R.

## Project Structure

- `data/`: Contains raw and processed data files.
- `codebook/`: Stores data dictionaries and codebooks describing the data.
- `reports/`: Destination for generated reports (e.g., HTML, PDF, Markdown).
- `scripts/`: R scripts for data analysis and visualization.

## Usage

Place your data in the `data/` directory and your analysis scripts in `scripts/`. Generated reports should be saved to `reports/`.
